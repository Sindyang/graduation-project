SYSTEM NAME: Umass_4D-costfidf
TDT TASK DESIGNATION

1) PRIMARY TEST SYSTEM DESCRIPTION: Cosine similarity
   Link detection comparing 2 stories using cosine similarity. Both story
   vectors were weighted using tf*IDF weighting scheme.
   
   Multiple databases:
     Global DB:   all docs machine translated into English.
     English DB:  only native English docs 
     Mandarin DB: only native Chinese docs
     Arabic DB:   only native Arabic docs

  For cross language pairs the similarity was computed using  the Global DB.
  Otherwise native language comparisons were done using the native language DB.

2) TRAINING:
   The threshold was determined by training on the newswire portion of the TDT4
   corpus. 
     Threshold: 0.0840
     Deferral:  10
