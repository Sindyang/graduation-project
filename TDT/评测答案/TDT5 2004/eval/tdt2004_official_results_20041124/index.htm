Topic Judgement files

The topic judgment files are available to all TDT 2004 participants. The LDC will distribute the topic annotations and the Link Detection key files through the LDC's e-Corpus menchanism. Contact Jon Fiscus if you do not automatically receive the topic judgements by Nov. 16, 2004.

Results

This release contains the results of the Topic Detection and Tracking 2004 Evaluation. There are 4 TDT 2004 evaluation tasks: Topic Tracking, Hierarchical Topic Detection, New Event Detection and Link Detection. Results of the supervised adaptive topic tracking are treated as a separate task an tabluated separately. There are two summaries for each of the four evaluations. The first set of four includes only the primary evaluation conditions, and the second set of four contains entries for all submissions.

The first three are topic sets defined by the size of topics, i.e., the number of ontopic stories that are within the test epoch. The three divisions are topics with between 1 and 10 (TS=gt0-lte10), 11 and 100 (TS=gt10-lte100), and more than 100 (TS=gt100) testable ontopic stories per topic. The fourth topic set consists of the multilingual topics, (TS=multiling), that have at least 1 testable ontopic story in each of the three languages.
